# Neural Network Tuning
## K-fold cross validation
@(Machine Learning By Andrew Ng)[机器学习]
K-fold cross validation与测试集无关，k通常取10，将训练集划分为10个部分，每10个部分再分为10份，9份训练，一份测试，训练10次，取训练后结果的平均值。用来避免某一次训练结果可能过高或者过低。考察两个指标，bias和variance，bias与准确度相关，即最终到底准不准确，variance表示最后的10次结果之间的差别大不大。
<!--more-->
具体代码如下：
``` python
from keras.wrappers.scikit_learn import KerasClassifier
from sklearn.model_selection import cross_val_score
from keras.models import Sequential
from keras.layers import Dense
def build_classifier():
    classifier = Sequential()
    classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu', input_dim = 11))
    classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu'))
    classifier.add(Dense(units = 1, kernel_initializer = 'uniform', activation = 'sigmoid'))
    classifier.compile(optimizer = 'adam', loss = 'binary_crossentropy', metrics = ['accuracy'])
    return classifier
classifier = KerasClassifier(build_fn = build_classifier, batch_size = 10, epochs = 100)
accuracies = cross_val_score(estimator = classifier, X = X_train, y = y_train, cv = 10, n_jobs = -1)
mean = accuracies.mean()
variance = accuracies.std()
```
KerasClassifier是用于包裹sklearn中的cross_val_score方法（这个方法用于做k-fold）的wrapper，build_classifier这个函数只是将前面构建网络的代码打包了。通过KerasClassifier我们得到了分类器，`cross_val_score`里面的cv表示k-fold的k，n_job = -1表示要使用所有的cpu，这样可以进行并行计算。
## Dropout Regularization
通过Dropout即是去掉某一层的部分神经元，让各个神经元之间的联系变得独立，一定程度上可以避免过拟合。   
``` python
from keras.layers import Dropout
classifier = Sequential()
classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu', input_dim = 11))
classifier.add(Dropout(p=0.1))

classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu'))
classifier.add(Dropout(p=0.1))

classifier.add(Dense(units = 1, kernel_initializer = 'uniform', activation = 'sigmoid'))
classifier.add(Dropout(p=0.1))


classifier.compile(optimizer = 'adam', loss = 'binary_crossentropy', metrics = ['accuracy'])
```
添加第一层在后面加上`classifier.add(Dropout(p=0.1))`即可，后面的层同样。如果p=0.1还是不能够避免过拟合，就试试p=0.2，一直增加直到没有过拟合。

## 使用GridSearchCV调参
GridSearchCV可以代替之前的`cross_val_score`方法，传入需要调节的参数和分类器，给定k，在训练集上fit即可。fit好了之后该对象会有一个`best_params`和`best_score_`属性可以获取到最佳参数和最优分数。   
``` python
# Tuning the ANN
from keras.wrappers.scikit_learn import KerasClassifier
from sklearn.model_selection import GridSearchCV
from keras.models import Sequential
from keras.layers import Dense
def build_classifier(optimizer):
    classifier = Sequential()
    classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu', input_dim = 11))
    classifier.add(Dense(units = 6, kernel_initializer = 'uniform', activation = 'relu'))
    classifier.add(Dense(units = 1, kernel_initializer = 'uniform', activation = 'sigmoid'))
    classifier.compile(optimizer = optimizer, loss = 'binary_crossentropy', metrics = ['accuracy'])
    return classifier
classifier = KerasClassifier(build_fn = build_classifier)
parameters = {'batch_size': [25, 32],
              'epochs': [100, 500],
              'optimizer': ['adam', 'rmsprop']}
grid_search = GridSearchCV(estimator = classifier,
                           param_grid = parameters,
                           scoring = 'accuracy',
                           cv = 10)
grid_search = grid_search.fit(X_train, y_train)
best_parameters = grid_search.best_params_
best_accuracy = grid_search.best_score_
```
